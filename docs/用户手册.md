# 顺序连点器 用户手册（中文）

## 1. 安装与启动

### 1.1 环境要求

- Python 3.10+
- Windows 10/11（推荐）；macOS/Linux 视系统权限而定

### 1.2 安装

```bash
python -m pip install -r requirements.txt
```

### 1.3 启动

```bash
python run.py
```

## 2. 坐标点管理

### 2.1 添加坐标点

方式一：手动输入精确坐标

1. 在 X/Y 输入框输入数值
2. 点击“添加”

方式二：获取当前鼠标位置

1. 点击“获取当前位置”
2. 确认 X/Y 自动填充
3. 点击“添加”

### 2.2 编辑坐标点

- 双击表格中的 X 或 Y 单元格，使用数字微调
- 输入范围会自动限制在当前屏幕可用范围内

### 2.3 删除坐标点

- 选中一行或多行后点击“删除”
- 弹出确认对话框后选择“是”完成删除

### 2.4 拖拽排序

- 选中行后按住拖动，可调整执行顺序
- 序号列会自动反映执行顺序

## 3. 连点控制

### 3.1 开始/暂停/停止

- 开始：按顺序点击所有坐标点
- 暂停：暂停当前连点过程，保留执行进度
- 继续：暂停后再次点击暂停按钮（按钮文案变为“继续”）恢复执行
- 停止：终止连点并重置进度

### 3.2 间隔时间（毫秒）

- 范围：100–5000ms，默认 500ms，步长 100ms
- 修改后立即对后续点击生效

## 4. 循环模式（可选）

- 启用循环：按设置的循环次数重复执行坐标点序列
- 无限循环：不限制循环次数
- 循环间隔：每轮完成后等待的时间（毫秒）
- 状态栏会显示当前循环进度

## 5. 全局热键（可选）

- 勾选“启用全局热键”后，可在后台用热键控制开始/暂停(继续)/停止
- 支持组合键（如 `ctrl+shift+s`）
- 开始/暂停/停止三组热键必须互不相同

## 6. 后台运行与托盘

- 最小化窗口或点击右上角关闭会驻留系统托盘
- 托盘菜单可显示/隐藏窗口，并可开始/暂停/停止/退出

## 7. 异常与排查

### 7.1 FailSafe 触发

鼠标移动到屏幕角落会触发 pyautogui FailSafe 并强制停止，用于防止失控连点。

### 7.2 分辨率变化导致越界

应用会在开始前校验坐标，运行中检测到坐标越界会安全停止并提示。

### 7.3 日志

应用会记录打开/保存/开始/暂停/停止/异常等操作到本机应用数据目录下的日志文件，便于排查问题。
